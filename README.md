# Stardust
Fractional nfts!


## running

Both the backend and frontend use npm. 

### frontend
1. `cd` into frontend folder.
2. run `npm i` to install dependencies.
3. run `npm start` to start the front-end.

### backend
1. `cd` into backend folder.
2. run `npm i` to install dependencies.
3. run `npm start` to start the server.
